# Bryansk cottages landing

Статический лендинг-отчет по домам и коттеджам у Брянска на 31.07-02.08.2026.

Плановый URL после публикации: `https://perf769.github.io/bryansk-cottages-2026/`.

## Публикация на GitHub Pages

1. Скопировать директорию `bryansk-cottages-2026` в корень репозитория GitHub Pages.
2. В настройках GitHub Pages выбрать ветку и папку с этим лендингом.
3. Открыть `https://perf769.github.io/bryansk-cottages-2026/`.

Сайт работает как статический отчет. Голосование в текущем виде сохраняет голос в `localStorage` браузера и подходит для демо или обсуждения. Для реального учета голосов см. `poll-backend.md`.

## Данные

- `data/analysis.json` - основной набор данных для PDF и сайта.
- `data/analysis.js` - JS-копия того же набора, чтобы сайт открывался без локального сервера.
- `assets/screenshots` - облегченные JPG-скриншоты карточек и проверок.
